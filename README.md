# Tech Challenge - Fase 2

## Sobre o projeto

Este repositório será utilizado para o trabalho acadêmico desenvolvido para o **Tech Challenge - Fase 2** da FIAP.

O projeto tem como tema a classificação da qualidade de vinhos com base em características físico-químicas, utilizando o **Wine Quality Dataset**.

## Objetivo

Desenvolver uma análise e um modelo de classificação para prever se um vinho pode ser considerado de alta qualidade.

A variável `quality` será transformada em uma classificação binária:

- **Alta qualidade:** vinhos com `quality >= 7`;
- **Baixa ou média qualidade:** vinhos com `quality < 7`.

## Estrutura do repositório

```text
wine-quality-classification/
+-- data/                 # Base de dados utilizada
+-- notebooks/            # Notebook com análise e modelagem
+-- src/                  # Scripts auxiliares
+-- results/              # Gráficos, métricas e resultados
+-- presentation/         # Apresentação executiva
+-- requirements.txt      # Bibliotecas utilizadas
+-- README.md             # Descrição do projeto
```

## Base de dados

A análise será desenvolvida com o **Wine Quality Dataset**, base pública com características físico-químicas de vinhos e notas de qualidade atribuídas por especialistas.

Fonte: [Kaggle - Wine Quality Dataset](https://www.kaggle.com/datasets/yasserh/wine-quality-dataset)
